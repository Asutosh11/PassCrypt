# Offline Password Manager v1.0.8 - Release Notes

Welcome to the initial release of **Offline Password Manager**, your ultra-secure, offline-first password manager. Built for those who value privacy above all else.

## Core Philosophy: Privacy by Design
Offline Password Manager was built on three non-negotiable pillars:
1. **100% Offline**: Your passwords never touch the internet. No syncing, no cloud exposure, no leaks.
2. **Zero Permissions**: We don't ask for your location, contacts, or internet access. The only permission we use is Biometrics for your security.
3. **Encrypted at Rest**: Every byte of your data is protected using **AES-256 GCM** encryption, managed by the Android Keystore system.

## Key Features
- **Professional Minimalist UI**: A clean, modern interface designed for speed and clarity.
- **Biometric Locking**: Unlock your passwords instantly using your device's fingerprint or face scan.
- **Adaptive Layout**: Optimized for both phones and tablets (7" and 10"). Tablet users enjoy a professional Master-Detail layout for efficient management.
- **Smart Categorization**: Group your credentials into Work, Personal, Finance, and more with vivid blue icons.
- **In-App Password Generator**: Create secure, random passwords without leaving the app.
- **Instant Search**: Real-time searching with a clean, high-performance interface.
- **Secure Password Transfer**:
- **QR Chunks**: Move small batches directly between Android devices, no cloud required.
- **Encrypted PDF Backups**: Save your passwords to a password-protected PDF for legacy recovery.
- **iOS / iCloud Keychain Export**: Dedicated CSV export compatible with Apple's password management.
- **App Rating**: Share your feedback with a single tap after using the app.

## Technical Specs
- **Database**: Encrypted SQLCipher database.
- **Encryption**: AES-256 GCM with hardware-backed keys.
- **Platform**: Native Android developed with Jetpack Compose.
- **Local Storage**: Data persists only on your physical device.
