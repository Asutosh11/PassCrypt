# Offline Password Manager - Play Store Description Draft

## Short Description (Max 80 characters)
Secure, 100% offline password manager with AES-256 encryption and biometric lock.

---

## Full Description

No servers. No accounts. No internet permission. No data collection. Your passwords stay encrypted on your device, fully offline.

## SECURITY

- AES-256 Encryption — Industry-standard protection
- Offline by Design — No internet permission. Ever.
- Hardware-Backed Keys — Secured by your device's biometric/PIN enclave
- Biometric Unlock — Fingerprint or face authentication
- Zero Tracking — No ads. No analytics. No telemetry.

## BACKUP & TRANSFER

- Encrypted QR / USB phone-to-phone transfer
- Password-protected PDF export
- iOS-compatible CSV export

## FEATURES

- Dark mode support
- Clean UI
- Custom categories
- Password generator
- High-performance search

## WHY OFFLINE PASSWORD MANAGER?

Most password managers store your data on remote servers, a centralized target for attackers. Offline Password Manager removes cloud sync entirely. No accounts. No subscriptions for core features. Just private, offline security.
